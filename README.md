# semver

Command line tool to create new git tags.

## Usage

```
Create semver releases

Usage:
  semver [flags]
  semver [command]

Available Commands:
  completion  generate the autocompletion script for the specified shell
  current     Show the current tag
  help        Help about any command
  update      Update the binary with a new version

Flags:
  -n, --dryrun    Dry run mode
  -h, --help      help for semver
  -M, --major     Perform a major release
  -m, --minor     Perform a minor release (default true)
  -p, --patch     Perform a patch release
  -s, --silent    Disable all output
  -v, --verbose   Increase verbosity

Use "semver [command] --help" for more information about a command.
```
